.. note::

    MPI is disabled by default.

    Use ``HYPRE_WITH_MPI=ON`` to enable it (and make sure MPI is installed on your system).

    .. code-block:: cmake

        # config.cmake
        hunter_config(hypre
            VERSION ${HUNTER_hypre_VERSION}
            CMAKE_ARGS
                HYPRE_WITH_MPI=ON
        )
